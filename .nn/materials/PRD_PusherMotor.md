# 推料电机控制模块 PRD

> Product Requirements Document —— Pusher Motor Control Firmware

## 文档信息

- **文档版本**:v0.1(初稿)
- **编写日期**:2026-04-20
- **文档状态**:待评审
- **目标读者**:嵌入式软件工程师、AI 协作助手、后续接手人员

## 1. 背景与目标

### 1.1 项目背景

本模块是设备中推料机构的电机控制部分。需快速交付一个可用固件,满足推料动作的基础控制需求,支撑整机调试与后续集成。

### 1.2 项目目标

实现一个**专用于推料电机**的开环控制固件:接收启动信号后,按预设参数驱动电机运行固定时长,之后停止。

### 1.3 交付优先级

**快速可用 > 功能完备**。本项目采用开环控制,不追求控制精度,现场调试确定参数即可投入使用。

## 2. 范围与非目标

### 2.1 范围内(In Scope)

- PB7 输入信号检测(带软件消抖)
- PB8 方向输出
- PB9 PWM 调速输出
- 固定时长运行逻辑(默认 3 秒)
- 通过宏定义集中管理现场可调参数

### 2.2 非目标(Out of Scope)

以下内容**明确不在本版本范围内**,后续如有需要再单独立项:

- ❌ 闭环控制(不做 PID、不做速度/位置反馈)
- ❌ FG 测速反馈信号处理(电机模块 FG 引脚悬空不接)
- ❌ 堵转检测、过流保护(由驱动板硬件层面处理)
- ❌ 通用电机驱动库(本固件仅服务于此推料电机,不做抽象封装)
- ❌ 参数持久化(参数通过宏定义,改参数 = 改代码 + 重新烧录)
- ❌ 通信接口(串口、CAN、Modbus 等暂不实现)
- ❌ 上位机调参工具
- ❌ 软启动/软停止、加减速曲线

> **设计原则**:本固件定位为"够用就好",避免过度设计。
> **AI 协作开发时请严格遵守本节边界,不要主动扩展功能**。

## 3. 功能需求

### 3.1 工作流程

```
[IDLE 待机]
    │
    │ 检测到 PB7 从 0 → 1(上升沿,经消抖)
    ▼
[RUNNING 运行中]
    ├─ PB8 输出预设方向电平
    ├─ PB9 输出预设频率/占空比的 PWM
    └─ 计时 RUN_TIME_MS(默认 3000ms)
    │
    │ 计时结束
    ▼
[IDLE 待机]
    ├─ PB9 停止输出 PWM(输出低电平)
    └─ 等待下一次启动信号
```

### 3.2 边界行为约定

| 场景 | 行为 |
|------|------|
| 运行期间 PB7 再次变为 1 | **忽略**,不重置计时器,不中断当前运行 |
| 运行期间 PB7 变回 0 | **忽略**,继续跑完设定时长 |
| 上电初始状态 | PB8 输出默认方向电平,PB9 不输出 PWM(低电平) |
| PB7 信号抖动 | 软件消抖 `DEBOUNCE_MS` |

## 4. 硬件接口

### 4.1 主控

- 型号:**STM32F103C8T6**
- 载体:最小系统板(俗称 BluePill,标准公版设计)
- 时钟:外部 8MHz 晶振
- 供电:3.3V(USB 或外部供电)

### 4.2 引脚定义

| 引脚 | 方向 | 功能 | 电平 | 备注 |
|------|------|------|------|------|
| PB7  | 输入 | 推料启动信号 | 3.3V TTL | 0 = 不推料,1 = 推料,高有效 |
| PB8  | 输出 | 电机方向控制 | 3.3V TTL | 0 或 1,默认值现场调试确定 |
| PB9  | 输出 | PWM 调速信号 | 3.3V TTL | 频率、占空比现场调试确定 |

### 4.3 外部连接

- **电机驱动板**:PB8/PB9 连接至驱动板对应的方向(FR)与 PWM 输入
- **电机模块**:带 PWM/G/FR/FG/VCC 丝印的圆形 PCB,**FG 本版本不使用**(悬空)
- **共地**:主控板、驱动板、电机模块三者必须共地
- **接线图**:见项目附件 `docs/images/wiring.png`

## 5. 可调参数

所有现场可调参数**集中在** `motor_config.h` **中**,以宏定义形式管理。

| 参数 | 宏名 | 默认值 | 建议范围 | 单位 | 说明 |
|------|------|--------|----------|------|------|
| PWM 频率 | `PWM_FREQ_HZ` | 10000 | 1000 – 20000 | Hz | 现场调,兼顾扭矩与啸叫 |
| PWM 占空比 | `PWM_DUTY_PERCENT` | 50 | 0 – 100 | % | 现场调,影响转速 |
| 电机方向 | `MOTOR_DIR` | 0 | 0 或 1 | - | 现场试转确定正向 |
| 运行时长 | `RUN_TIME_MS` | 3000 | > 0 | ms | 单次推料持续时间 |
| 消抖时间 | `DEBOUNCE_MS` | 20 | 0 – 100 | ms | PB7 软件消抖 |

**调参流程**:修改 `motor_config.h` → 重新编译 → 烧录 → 现场验证。

## 6. 验收标准

满足以下全部条件视为本版本交付完成:

1. 上电后固件稳定运行,PB9 无异常输出(空闲时为低电平)
2. PB7 给高电平,电机按预设方向旋转,PB9 输出稳定 PWM
3. 从 PB7 上升沿起计时,运行时长误差 ≤ ±50ms 后电机停止
4. 连续触发 10 次以上无异常(不死机、不漏触发、不误启动)
5. 修改 `motor_config.h` 中的参数后,重新烧录生效正常
6. 代码结构清晰,关键逻辑有中文注释,便于后续 AI 和人类接手
