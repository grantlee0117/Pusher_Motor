# 推料电机控制模块 PRD

> Product Requirements Document —— Pusher Motor Control Firmware

## 文档信息

- **文档版本**: v1.0
- **编写日期**: 2026-04-20
- **最后更新**: 2026-04-24
- **文档状态**: 已交付
- **目标读者**: 嵌入式软件工程师、AI 协作助手、后续接手人员

## 1. 背景与目标

### 1.1 项目背景

本模块是设备中推料机构的电机控制部分。需快速交付一个可用固件,满足推料动作的基础控制需求,支撑整机调试与后续集成。

### 1.2 项目目标

实现一个**专用于推料电机**的开环控制固件：接收启动信号后,按预设参数驱动电机运行固定时长,之后停止。同时提供串口命令行接口（CLI）用于实时调参和调试，参数支持断电保存。

### 1.3 交付优先级

**快速可用 > 功能完备**。本项目采用开环控制,不追求控制精度,现场调试确定参数即可投入使用。

## 2. 范围与非目标

### 2.1 范围内(In Scope)

- PB6 输入信号检测（外部中断，上升沿触发）
- PB7 / PB8 方向输出（双路方向控制）
- PB9 PWM 调速输出
- 固定时长运行逻辑（默认 250 ms）
- 启动前等待延迟（默认 250 ms）
- 通过串口 CLI 实时调整参数
- 参数持久化到 Flash，断电不丢失
- 速度与 PWM 占空比换算（基于电机直径 6cm）

### 2.2 非目标(Out of Scope)

以下内容**明确不在本版本范围内**,后续如有需要再单独立项:

- ❌ 闭环控制(不做 PID、不做速度/位置反馈)
- ❌ FG 测速反馈信号处理(电机模块 FG 引脚悬空不接)
- ❌ 堵转检测、过流保护(由驱动板硬件层面处理)
- ❌ 通用电机驱动库(本固件仅服务于此推料电机,不做抽象封装)
- ❌ 软启动/软停止、加减速曲线
- ❌ 多电机同步控制

> **设计原则**:本固件定位为"够用就好",避免过度设计。
> **AI 协作开发时请严格遵守本节边界,不要主动扩展功能**。

## 3. 功能需求

### 3.1 工作流程

```
[IDLE 待机]
    │
    │ 检测到 PB6 上升沿（外部中断）或 CLI 发送 start 命令
    ▼
[WAIT 等待中]
    ├─ 延时 WAIT_TIME_MS（默认 250ms）
    │
    │ 等待结束
    ▼
[RUNNING 运行中]
    ├─ PB7/PB8 输出预设方向电平
    ├─ PB9 输出预设占空比的 PWM
    └─ 计时 DIRECTION_TIME_MS（默认 250ms）
    │
    │ 计时结束
    ▼
[STOP 停止中]
    ├─ PB9 停止输出 PWM（占空比恢复 500）
    └─ 清除工作标志
    │
    ▼
[IDLE 待机]
    └─ 等待下一次启动信号
```

### 3.2 边界行为约定

| 场景 | 行为 |
|------|------|
| 运行期间再次收到启动信号 | **忽略**,不重置计时器,不中断当前运行 |
| 运行期间 PB6 变回 0 | **忽略**,继续跑完设定时长 |
| 上电初始状态 | 从 Flash 加载参数，初始化方向引脚，PWM 输出占空比 500（停止） |
| 参数设置超出范围 | CLI 返回错误提示，拒绝保存 |

## 4. 硬件接口

### 4.1 主控

- **型号**: STM32F103C8T6
- 载体: 最小系统板(俗称 BluePill,标准公版设计)
- 时钟: 外部 8MHz 晶振，系统时钟 72MHz
- 供电: 3.3V(USB 或外部供电)

### 4.2 引脚定义

| 引脚 | 方向 | 功能 | 电平 | 备注 |
|------|------|------|------|------|
| PB6 | 输入 | 推料启动信号 | 3.3V TTL | 上升沿触发，外部中断 |
| PB7 | 输出 | 电机方向控制 B | 3.3V TTL | 0 或 1，默认 0 |
| PB8 | 输出 | 电机方向控制 A | 3.3V TTL | 0 或 1，默认 1 |
| PB9 | 输出 | PWM 调速信号 | 3.3V TTL | TIM4_CH4，占空比 0-500 |
| PA9 | 输出 | USART1_TX | 3.3V TTL | 调试串口发送 |
| PA10 | 输入 | USART1_RX | 3.3V TTL | 调试串口接收 |

### 4.3 外部连接

![硬件接线图](.nn/materials/2.png)

- **电机驱动板**: PB7/PB8 连接至驱动板对应的方向(FR)输入，PB9 连接 PWM 输入
- **电机模块**: 带 PWM/G/FR/FG/VCC 丝印的圆形 PCB, **FG 本版本不使用**(悬空)
- **启动信号**: PB6 连接上游设备的启动信号输出（高电平有效）
- **共地**: 主控板、驱动板、电机模块三者必须共地

## 5. 可调参数

> 实际测量设置电机的PWM占空比的范围30-500，如果设置占空比<30电机不能正常转动。

所有现场可调参数**通过串口 CLI 实时修改**，自动保存到 Flash，断电不丢失。

| 参数 | CLI 命令 | 默认值 | 范围 | 单位 | 说明 |
|------|----------|--------|------|------|------|
| 运行时长 | `set direction_time <ms>` | 250 | 1 – 60000 | ms | 单次推料持续时间 |
| PWM 占空比 | `set pwm_duty <value>` | 150 | 0 – 500 | - | 0=最快，500=停止 |
| 等待时间 | `set wait_time <ms>` | 250 | 0 – 10000 | ms | 启动前延迟 |
| 最高转速 | `set max_speed <rpm>` | 3655 | 1 – 10000 | RPM | 用于速度换算 |
| 目标速度 | `set speed <cm/min>` | - | 0+ | cm/min | 自动换算为占空比 |
| 电机A方向 | `set motor_mp_a_dir <0\|1>` | 1 | 0 或 1 | - | PB8 电平 |
| 电机B方向 | `set motor_mp_b_dir <0\|1>` | 0 | 0 或 1 | - | PB7 电平 |

**调参方式**: 通过串口发送 CLI 命令，参数自动保存到 Flash，无需重新编译烧录。

**临时调试**: 使用 `set new_pwm_duty <value>` 可直接修改 PWM 占空比，不保存到 Flash。

## 6. 验收标准

满足以下全部条件视为本版本交付完成:

1. 上电后固件稳定运行,PB9 无异常输出(空闲时占空比 500)
2. PB6 给上升沿信号,电机按预设方向旋转,PB9 输出稳定 PWM
3. 从启动信号起计时,等待 WAIT_TIME_MS 后运行 DIRECTION_TIME_MS，时长误差 ≤ ±50ms 后电机停止
4. 连续触发 10 次以上无异常(不死机、不漏触发、不误启动)
5. 通过串口 CLI 可正常修改所有参数，参数断电后保持
6. 代码结构清晰,关键逻辑有中文注释,便于后续 AI 和人类接手
