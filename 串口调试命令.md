# 串口调试命令文档

## 一、概述

本系统支持通过串口1（USART1）进行调试和控制，采用轮询模式进行数据收发。所有命令以换行符（`\n`）结尾。

## 二、串口配置

| 参数  | 值      |
| --- | ------ |
| 波特率 | 115200 |
| 数据位 | 8      |
| 停止位 | 1      |
| 校验位 | 无      |
| 流控制 | 无      |

## 四、命令列表

### 1. 启动推料电机

**命令**：`start`

**功能**：启动推料电机开始运行

**示例**：

```
start
```

**响应**：

```
Pusher motor started.
>
```

***

### 2. 设置运行时间

**命令**：`set direction_time <ms>`

**功能**：设置推料电机的运行时间

**参数**：

- `ms`：运行时间，单位为毫秒

**示例**：

```
set direction_time 2500
```

**响应**（成功）：

```
Direction time set to 2500 ms (saved to Flash)
>
```

**响应**（参数错误）：

```
Error: Invalid value. Range: 1-60000 ms
>
```

***

### 3. 获取运行时间

**命令**：`get direction_time`

**功能**：获取当前设置的运行时间

**示例**：

```
get direction_time
```

**响应**：

```
Current direction time: 250 ms
>
```

***

### 4. 设置PWM占空比

**命令**：`set pwm_duty <value>`

**功能**：设置PWM占空比，用于控制电机速度

**参数**：

- `value`：PWM占空比值，范围 0-500
- 注意：电机的PWM逻辑是电平越高速度越慢

**示例**：

```
set pwm_duty 250
```

**响应**（成功）：

```
PWM duty set to 250 (saved to Flash)
>
```

**响应**（参数错误）：

```
Error: Invalid value. Range: 0-500
>
```

***

### 5. 获取PWM占空比

**命令**：`get pwm_duty`

**功能**：获取当前设置的PWM占空比

**示例**：

```
get pwm_duty
```

**响应**：

```
Current PWM duty: 150
>
```

### 10. 设置速度

**命令**：`set speed <cm/min>`

**功能**：设置电机的速度（cm/分钟），系统会自动计算对应的PWM占空比

**参数**：

- `cm/min`：速度，单位为厘米/分钟，非负数

**示例**：

```
set speed 60000
```

**响应**（成功）：

```
Speed set to 60000 cm/min (PWM duty: 250) (Range: 0-68890 cm/min) (saved to Flash)
>
```

**响应**（缺少参数）：

```
Error: Missing value. Usage: set speed <cm/min> (Range: 0-68890 cm/min)
>
```

**响应**（参数错误）：

```
Error: Invalid value. Must be non-negative (Range: 0-68890 cm/min)
>
```

---

```

>
```

### 16. 直接设置PWM占空比（不保存）

**命令**：`set new_pwm_duty <value>`

**功能**：直接设置PWM占空比，**不保存到Flash**（用于临时调试）

**参数**：

- `value`：PWM占空比值，范围 0-500

**示例**：

```
set new_pwm_duty 300
```

**响应**（成功）：

```
PWM duty directly set to 300 (not saved to Flash)
>
```

**响应**（参数错误）：

```
Error: Invalid value. Range: 0-500
>
```

**注意**：此命令与 `set pwm_duty` 的区别：
- `set pwm_duty`：设置占空比并保存到Flash，断电后保持
- `set new_pwm_duty`：仅临时修改占空比，不保存到Flash，断电后恢复

***

### 17. 设置加速度步长

**命令**：`set acceleration <0-50>`

**功能**：设置电机启动时的加速度步长，控制从停止到目标速度的加速过程

**参数**：

- `0`：关闭加速，直接输出目标占空比（默认）
- `1-50`：加速步长，每隔 10ms 占空比减少该值（值越大加速越快）

**示例**：

```
set acceleration 10
```

**响应**（成功）：

```
Acceleration set to 10 (0=off)
>
```

**响应**（参数错误）：

```
Error: Invalid value. Range: 0-50
>
```

**注意**：
- PWM 逻辑中占空比 500 = 停止，0 = 最高速
- 加速过程：从占空比 500 开始，每隔 10ms 减去 `acceleration` 值，直到达到目标占空比
- 该参数为运行时参数，不保存到 Flash（重启后恢复默认值 0）

***

## 五、参数存储

所有通过 `set` 命令设置的参数都会自动保存到Flash存储器中，断电后参数不会丢失。系统启动时会自动从Flash加载之前保存的参数。

## 六、注意事项

1. **命令格式**：命令和参数之间使用空格分隔
2. **换行符**：所有命令必须以换行符（`\n`）结尾
3. **参数范围**：请确保参数在有效范围内，否则会返回错误信息
4. **响应延迟**：由于使用轮询模式，响应可能会有轻微延迟
5. **PWM逻辑**：电机的PWM逻辑是电平越高速度越慢，设置时请注意
