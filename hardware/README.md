# Hardware

本目录集中保存硬件资料。这里的文件用于团队查看原理图、接口定义、板载连接器位置和后续硬件改版信息。

当前纳入的是本次计划替换的新板（自研一体板）资料；它不表示当前固件已经完成新板适配。固件是否适配仍以 `Pusher_Motor.ioc`、`Core/` 外设初始化和 `App/` 业务代码中的实际配置为准。

## 目录结构

```text
hardware/
└── new-pcb-board/               # 后续新板资料
    ├── board-view/              # 板面、接口位置和标注图
    ├── interface/               # IO 分配表和导出截图
    └── schematic/               # 原理图源文件
```

## 维护约定

- 原理图源文件放在 `schematic/`。
- 接口表、引脚表和从表格导出的图片放在 `interface/`。
- 板面截图、接口位置标注图、装配参考图放在 `board-view/`。
- 文件名尽量包含用途、版本或日期，避免只使用“截图”这类临时名称。
- 硬件资料更新后，需要同步检查 `docs/` 中的接口说明是否还准确。
- 后续真正做固件适配时，再单独修改 `Pusher_Motor.ioc`、`Core/`、`App/` 等代码相关文件。
